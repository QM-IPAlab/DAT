The Detector Teaches Itself:
Lightweight Self-Supervised Adaptation for Open-Vocabulary Detection

Yazhe Wan1, Changjae Oh1
1Queen Mary University of London
Paper arXiv Code
Figure 1. We bridge the Global-Local feature gap in VLMs. Our method (DAT) adapts the model using self-generated pseudo-labels, significantly improving novel object detection.

Abstract

We address a critical bottleneck in cooperative open-vocabulary object detection: the Global–Local Feature Imbalance in pre-trained Vision-Language Models (VLMs). While VLMs excel at whole-image understanding, they struggle with cropped region proposals during detection.

To bridge this gap, we introduce Decoupled Adaptivity Training (DAT), a self-supervised fine-tuning paradigm that efficiently adapts VLMs to region-level recognition without compromising their zero-shot capabilities. DAT operates exclusively on the visual backbone using in-pipeline pseudo-labels from a frozen detector, constructing a perfectly domain-aligned training set.

Through a novel combination of a sigmoid-based contrastive loss for robust multi-label learning and WiSE-FT for global knowledge preservation, DAT achieves significant improvements in novel-class detection while maintaining strong base-class performance—all with minimal computational overhead.

Methodology

The DAT framework consists of two stages: Region-Aware Data Construction & Decoupled Fine-Tuning.

Figure 2. Overview of the DAT methodology. We decoupled VLM visual backbone in fine-tuning phase while freeze other parameters.

1. Region-Aware Data

We construct a domain-aligned dataset $\mathcal{D}_{\text{region}}$ by cropping object proposals. Using $N_{\text{max}}=80$ and $\tau_{\text{conf}}=0.7$, we generate ~400K high-quality region-text pairs from just 5K images.

2. Decoupled Fine-Tuning

We fine-tune only the visual encoder. To prevent catastrophic forgetting, we use WiSE-FT with $\alpha=0.5$, blending the fine-tuned weights with the original zero-shot weights.
